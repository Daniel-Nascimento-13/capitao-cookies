
# Projeto "Capitão Cookies"

Este repositório reúne *uma landing page*, voltada para a divulgação da marca *Capitão Cookies*, que já está consolidada há mais de *5 anos no mercado*.

O projeto foi criado com o objetivo de *causar uma excelente primeira impressão*, apresentar os produtos de forma atrativa e *potencializar a geração de leads*.

A landing page foi desenvolvida para conectar o cliente diretamente ao atendimento, oferecendo uma experiência *prática, rápida e envolvente*, além de simular uma pequena loja sem sistema de pagamento integrado, sendo um projeto *ativo e em uso real*, e não apenas uma simulação.

---

- **Landing page**

👉 https://capitaocookies.com.br

---

O site é *destinado* a pessoas que desejam conhecer os produtos da marca, visualizar as opções de cookies disponíveis e realizar seus pedidos de forma simples, sendo direcionadas para o *WhatsApp*, onde o atendimento é finalizado de forma personalizada.
